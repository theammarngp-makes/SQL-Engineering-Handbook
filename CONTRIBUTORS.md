## How to Get Featured

Every contribution matters! To be featured as a contributor:

1. **Submit a Pull Request** with your contribution
2. **Include a clear commit message** explaining what you added
3. **Once merged**, you'll automatically appear in our Contributors section

### 🤝 Community Contributors

**We're using the [All Contributors](https://allcontributors.org/) format to recognize all contributions.**

Once you contribute, add yourself to the table below by including this comment in your PR:

```
@all-contributors please add @your-github-username for [type of contribution]
```

**Contribution types:**
- 🐛 `bug` - Bug fixes
- 💻 `code` - Code/query contributions
- 📖 `doc` - Documentation improvements
- 🤔 `ideas` - Ideas & suggestions
- ✏️ `content` - Writing content
- 🌍 `translation` - Translations (SQL dialects)
- 🔧 `maintenance` - Maintenance & optimization
- 🔍 `review` - Code review
- 📢 `promotion` - Promotion & sharing

## Want to Become a Contributor?

### Easy Ways to Start:

**1. Add a Single Query (30 min)**
- Pick a section
- Write one query with explanation
- Submit a PR

**2. Translate One Query (20 min)**
- Pick a query
- Translate to Snowflake/BigQuery/PostgreSQL/T-SQL
- Submit a PR

**3. Report a Bug (5 min)**
- Find an issue
- Open a bug report issue
- Help us improve!

**4. Suggest a Feature (5 min)**
- Have an idea?
- Open a feature request
- Let's discuss!

**See CONTRIBUTING.md for detailed guidelines** → [CONTRIBUTING.md](CONTRIBUTING.md)
