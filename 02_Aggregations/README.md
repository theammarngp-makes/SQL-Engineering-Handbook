# SQL Aggregations

This section covers SQL aggregate functions used to summarize, analyze, and report data.

## Topics Covered

| No. | Topic |
|------|--------|
| 1 | COUNT() |
| 2 | SUM() |
| 3 | AVG() |
| 4 | MIN() & MAX() |
| 5 | GROUP BY |
| 6 | HAVING |

## Learning Goals

After completing this section, you should be able to:

- Count records
- Calculate totals
- Calculate averages
- Find minimum and maximum values
- Group records
- Filter grouped results
