# SQL Fundamentals

This section covers the core SQL concepts required before learning joins, aggregations, subqueries, and advanced analytics.

## Topics Covered

| No. | Topic |
|------|--------|
| 1 | SELECT |
| 2 | WHERE |
| 3 | ORDER BY |
| 4 | LIMIT |
| 5 | ALIAS |

---

## Learning Goals

After completing this section, you should be able to:

- Retrieve data using SELECT
- Filter records using WHERE
- Sort data using ORDER BY
- Remove duplicates using DISTINCT
- Limit result sets using LIMIT
- Improve readability using ALIAS
